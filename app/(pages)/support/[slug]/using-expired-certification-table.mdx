# Using the Expiring Certifications Table 


# Objective

To use the expring certification table effectively.

# Steps
1. Navigate to the bottom of the **Employee Dashboard.**
2. Employees listed in this table have one or more expired certifications, or upcoming certifications that are to expire within the timeframe set; update the timeframe at the top of the table by typing in how many days out you would like the table to filter to.  
*(For Example: 30 days, 60 days, 90 days.)*
3. In the far-right column, you may **View Employee, View Image, Add Certification.**

![Actions for employee on the certifications table](/public/Images/support-page-screenshots/accoric-management-certification-options.webp)

### <ins>Grouping the Table</ins>

1. The following image displays the three types of groupings:

![Grouping Types for the certifications Table](/public/Images/support-page-screenshots/accoric-management-certification-groupings.webp)

2. **Default View:** This will display employees' name and the certification they hold that is expiring. in this view, a line will be added for each certification that is expiring for that employee.  
*(For example, if Matthew holds two certifications that are expiring - OSHA 30 in Construction, Scissor Lift Operator Certification - then two lines will be added to the table.)*  
Default View is shown in this image.
3. **Employee View:** This will condense all expired certifications to each employee. Clicking **view** will show that employee's expired certifications in a table you may scroll through.
4. **Certification View:** This view will prompt the user to sort the table from a list of certifications in the dropdown. Certifications listed here are pulled from certification names you previously uploaded for employees.  
*For Example, if you upload a certification listed here are pulled from certification names you previously uploaded for employees. For Example, if you upload a certification names "Forklift Operator," for an employee, that name will appear in the dropdown.*  
Selecting a certification type from this dropdown will filter the table to employees that are holding a certification of taht type that is expired.
